Posted on 2019-07-03 In Technology

Unity面试题:

一:：什么是协同程序?

在主线程运行时同时开启另一段逻辑处理，来协助当前程序的执行，协程很像多线程但不是多线程，Unity的协程在每帧结束之后去检测yield的条件是否满足

二：ArrayList与List的主要区别？

​ ArrayList存在不安全类型（ArrayList会把所有插入其中的数据都当作Object来处理）装箱拆箱的操作（费时）。List是接口，Array是实现了一个实现了该接口的类，可以被实例化。

三：请简述sealed（seal密闭的过去分词和过去式）关键字用在类声明时与函数声明时的作用

​ sealed修饰的类为密封类，类声明时可以防止其他类继承该类，在方法中声明则可防止派生类重写此方法。

四：Unity生命周期：

Awake -> OnEnable -> Start ->Update -> FixedUpdate -> LateUpdate -> OnGUI -> OnDisable -> OnDestory

五：物理函数一般放在FixedUpdate里

​ 每固定帧绘制执行一次，和Update不同的时FixedUpdate是渲染帧执行，如果你的渲染效率低下，FixedUpdate调用次数也跟着下降。FixedUpdate比较适用于物理引擎的计算，因为是跟每帧渲染有关。Update就比较适合做控制。

六：LOD是什么，优缺点又是什么？

​ LOD(Level of detail)多层次细节。按照模型位置和重要程度决定物体渲染的资源分配，降低非重要物体的面数和细节度，从而获得高效率的渲染运算。缺点是增加内存。

七：MipMap是什么，作用？

​ MipMapping:在三维计算机图形的贴图渲染中常用的技术，为加快渲染进度和减少图像锯齿，贴图被处理成由一系列预先计算和优化过的图片组成的技术，这样的贴图叫做MipMap

八：Interface和抽象类之间的不同

​ 抽象类表示该类中可能已有一些方法的具体定义，但接口就是公共只能定义各个方法的界面，不能具体地实现代码在成员方法中。类是子类用来继承的，当父类已经有实际功能的方法时该方法在子类中可以不必实现，直接引用父类的方法，子类也可以重写该父类方法。实现接口的时候必须要实现接口中所有的方法，不要遗漏。
